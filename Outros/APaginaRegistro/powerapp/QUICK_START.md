## ⚡ 5-Minute Crash Course

### 1. Design Tokens (Cores)
```yaml
# Defina UMA VEZ no topo

$Color_Primary: =RGBA(0, 120, 212, 1)
$Color_Error: =RGBA(168, 0, 0, 1)

# Use em TODOS os componentes
Fill: =$Color_Primary
BorderColor: =$Color_Error
```

### 2. Nomeação
```yaml
# Padrão: Card[NomeDoCampo]_[SubComponente]

- CardTítulo
  - CardTítulo_Label
  - CardTítulo_Input
  - CardTítulo_ErrorMsg
  - CardTítulo_RequiredIndicator
```

### 3. Documentação
```yaml
- CardTítulo:
    Description: "Campo para entrada do título"
    # ← Sempre adicione descrição
```
